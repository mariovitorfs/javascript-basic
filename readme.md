Criada em 1995 por Brendan Eich, para ser usada pelo netscape, o nome original é ECMAScript.
é uma linguagem de alto nivel e interpretada multiparadigma.

## Tipos de dados
- Tipos primitivos:

number e bigint = Números
string = palavras
boolean = Verdadeiro ou falso (true or false) // 1 ou 0
null = Vazio
undefined =  indefinido (não tem valor atribuido)

bigint é para numeros muito grandes, sendo utilizados com "n" no final do numero.
Ex: 199999394394934934n

## Variáveis

Servem para armazenar os dados para podermos usá-los no nosso código.

Temos 3 opções para criar variaveis:

Var = Cria variavel que pode ser alterada.
let = é uma forma mais moderna de declarar variaveis. sendo usada em blocos de códigos.
const = Cria variavel que não pode ser alterada.

Nomeando variáveis:

é uma boa pratica de programação criar variaveis com nome apropriado para as variaveis, com nome que fazem sentido pro tipo de variavel criado. 

existem cases de nomeclatura para facilitar o nome da variavel

ex:

camel case: nomeDeVariavel

snake case: nome_de_variavel


## Operadores e expressões

Aritméticos: +, -, *, / e %

Agrupamento ()

Atribuição =, +=, -=, *=, /= e %=

Encadeamento = "é o . usado ex: console.log"

Incrementar ++ e Decrementar -- "se o ++/-- estivar antes da variavel, ele retorna o valor atualizado."
